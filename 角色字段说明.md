# 📋 角色信息字段说明

## 角色信息字段结构

### 完整字段列表

```json
{
    "Name": "角色姓名",
    "Gender": "性别",
    "Evaluation": "角色简介",
    "Intro": "详细描述",
    "FirstMsg": "开场白",
    "Categories": ["标签1", "标签2", "标签3"],
    "SoundTags": ["声音特征1", "声音特征2"],
    "Scene": "对话场景",
    "DialogExample": "对话示例"
}
```

---

## 字段详细说明

### 1. Name（姓名）
- **类型**：字符串
- **说明**：角色的名字
- **示例**：`"小樱"`、`"Alex"`、`"艾莉娅"`

### 2. Gender（性别）
- **类型**：字符串
- **取值**：`男` / `女` / `其他`
- **说明**：角色的性别
- **示例**：`"女"`

### 3. Evaluation（简介）
- **类型**：字符串
- **长度**：1-2 句话
- **说明**：角色的简短介绍，突出核心特点
- **示例**：`"活泼开朗的高中生，喜欢画画和音乐"`

### 4. Intro（角色描述）
- **类型**：字符串
- **长度**：3-5 句话
- **说明**：角色的详细描述，包括：
  - 外观特征
  - 性格特点
  - 背景故事
  - 兴趣爱好
- **示例**：
```
小樱是一名16岁的高中二年级学生，有着粉色长发和蓝色眼睛。
性格活泼开朗，总是充满活力，喜欢结交新朋友。
她对艺术充满热情，特别喜欢画画和听音乐，梦想成为一名插画师。
```

### 5. FirstMsg（开场白）
- **类型**：字符串
- **长度**：30-50 字
- **说明**：角色第一次见面时会说的话
- **要求**：
  - 要符合角色性格
  - 要有个性和特点
  - 要让用户感到亲切
- **示例**：
  - 活泼型：`"嗨！初次见面，我是小樱！很高兴认识你~ 今天天气真好呢！"`
  - 温柔型：`"你好...我是艾莉娅。能认识你真好，请多关照。"`
  - 冷酷型：`"...你找我有事？"`

### 6. Categories（角色标签）
- **类型**：数组（字符串列表）
- **数量**：4-6 个标签
- **说明**：描述角色的多维度标签
- **分类**：
  - 身份标签：学生、教师、医生、战士等
  - 性格标签：活泼、温柔、高冷、开朗等
  - 风格标签：二次元、写实、科幻、奇幻等
  - 特征标签：猫娘、精灵、机器人等
- **示例**：
```json
["高中生", "活泼", "艺术爱好者", "二次元", "少女", "乐观"]
```

### 7. SoundTags（声音标签）
- **类型**：数组（字符串列表）
- **数量**：2-3 个标签
- **说明**：描述角色的声音特征
- **常用标签**：
  - 音色：温柔、清脆、低沉、磁性、甜美、沙哑
  - 语气：活泼、平静、严肃、俏皮、慵懒
  - 特点：御姐音、萝莉音、少年音、成熟
- **示例**：
  - `["清脆", "活泼"]`
  - `["温柔", "甜美"]`
  - `["低沉", "磁性"]`

### 8. Scene（对话场景）
- **类型**：字符串
- **说明**：角色常出现的对话场景或环境
- **示例**：
  - `"学校教室、美术室、公园"`
  - `"咖啡厅、书店、家中客厅"`
  - `"战场、训练场、军事基地"`
  - `"魔法学院、图书馆、森林"`

### 9. DialogExample（对话示例）
- **类型**：字符串
- **格式**：`角色名：对话内容`
- **说明**：展示角色说话风格和特点的对话示例
- **要求**：
  - 要符合角色性格
  - 要展示说话习惯
  - 可以包含口头禅
  - 要自然生动
- **示例**：
```
小樱：哇，你也喜欢画画吗？太好了！我最近在练习水彩，你要不要一起来画画呀？

艾莉娅：今天的月亮真美呢...你知道吗，每当我看到月亮，就会想起家乡的样子。

雷克斯：哼，这种程度的训练根本不算什么。想要变强，就必须突破极限！
```

## 🎨 生图提示词规范

### 严格要求

1. ✅ **必须使用英文**
2. ✅ **只用关键词或简短短语**
3. ❌ **禁止长篇修饰**
4. ❌ **禁止小说化描述**

### 正确格式

```
masterpiece, best quality, 1girl, anime style, long pink hair, blue eyes, cat ears, white dress, gentle smile, standing
|||
lowres, bad anatomy, bad hands, text, error, missing fingers, extra digit, cropped, worst quality, low quality
```

### 错误示例 ❌

```
A beautiful girl with flowing pink hair that cascades down her shoulders like a waterfall, her crystal blue eyes sparkling with joy...
```

### 正确示例 ✅

```
1girl, long pink hair, blue eyes, white dress, gentle smile
```

## 🔧 使用说明

这些字段会在角色创建过程中通过对话逐步收集：

1. **Name & Gender**：最基础的信息，首先询问
2. **Evaluation & Intro**：通过询问性格、背景等收集
3. **FirstMsg**：根据角色性格生成或用户自定义
4. **Categories**：根据收集的信息自动提取标签
5. **SoundTags**：询问用户对声音的期望
6. **Scene**：询问角色的活动场景
7. **DialogExample**：根据说话风格生成示例

Agent 会智能地通过 3-5 轮对话自然收集这些信息！🎉
